# New-Spell

## SYNOPSIS
Create a new spell for your PC or NPC to use/discover.

## SYNTAX

```
New-Spell [[-Mutation] <Boolean>] [[-Omen] <Boolean>] [<CommonParameters>]
```

## DESCRIPTION
Creates a spell.

## EXAMPLES

### EXAMPLE 1
```
$spell = New-Spell
PS C:\> $spell
```

PhysicalEffect  :
PhysicalElement :
PhysicalForm    : Trap
EtherealEffect  :
EtherealElement : Warp
EtherealForm    :
Mutation        :
Insanity        :
Omen            :

PS C:\\\> $spell.ToString()
Warp Trap

Create a new spell and save it to a variable.
